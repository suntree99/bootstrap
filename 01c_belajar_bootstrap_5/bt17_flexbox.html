Flexbox

Flex Container adalah div pembungkus dalam implementasi flexbox.
Flex Item adalah div item di dalam Container.
Content Item adalah konten di dalam Flex Item.

Flex Display and Direction

Display Flex (d-flex).
Display Inline Flex (d-inline-flex). Memaksa Container inline (di kiri).
Flex Row (flex-row) dari kiri ke kanan | Flex Row Reverse (flex-row-reverse) sebaliknya.
Flex Column (flex-row) dari atas ke bawah | Flex-column-reverse (flex-column-reverse) sebaliknya.

Display Flex (d-flex) dan Flex Item

Lebar Flex Item akan menyesuaikan lebar Content Item (text).
Jika Flex Item banyak, lebarnya akan menyesuaikan sehingga tetep muat dalam Containernya.
Tinggi Flex Item akan memenuhi tinggi Containernya (default: align-item-stretch) kecuali jika ditentukan.
Arah Flex Item adalah dari kiri ke kanan (default: flex-row).
